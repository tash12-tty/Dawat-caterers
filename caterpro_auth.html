📊 Database Info

All orders are stored in Firebase Firestore under your dawat-caterers project. Customer orders arrive automatically from the Dawat Caterers website. You can also add orders manually using the + New Order button.

To manage or view raw data, visit console.firebase.google.com.

🤖 AI Prioritization

Use ✦ AI Rate on any order card to rate a single order, or use ✦ AI Prioritize All in the topbar to rate every active order at once. Priorities are saved back to Firebase instantly.

⚠ Danger Zone

Deleting orders is permanent and cannot be undone. Delete individual orders using the 🗑 button on each order card.
